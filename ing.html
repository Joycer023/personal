ING

ING

ING is een van de grootste banken in Nederland en heeft zich gevestigd in verschillende landen. ING heeft een in-house IT bedrijf. Dit bedrijf zorgt voor de IT van de ING Groep. Het bedrijf wilt erg innovatief zijn op het gebied van IT en services. Wereldwijd zijn voor het bedrijf meer dan 50.000 medewerkers in dienst.

It-rollen


Virtual assistent. (n.d.). [Illustration]. https://www.elegantthemes.com/blog/wp-content/uploads/2020/02/become-a-virtual-assistant-featured-image.jpg

C# DEV ENGINEER
Als C# DEV ENIGEER hou je je vooral bezig met het designen ontwikkelen en testen van software oplossingen en applicaties. De programmeertaal die je gebruikt is voornamelijk C#. Ook houdt je je bezig met het ondersteunen van het team.

FRAUD & IT SECURITY OFFICER
Als FRAUD & IT SECURITY OFFICER hou je je bezig met fraude en IT security. Zo ben je bijvoorbeeld bezig met het opvolgen van dreidingsinformatie en testresultaten van het Red en Blue-team, maar ook bepaal je de maatregelen voor technische processen die zich bezig houdt met fraude.

SQL SERVER DEVELOPER
De taken van een SQL SERVER DEVELOPPER zijn het testen en automatiseren van oplossingen in de SQL Server, veranderen van data in de database, etc… Als SQL SERVER DEVELOPER ben je dus eigenlijk continue bezig met SQL.

CUSTOMER JOURNEY EXPERT – VIRTUAL ASSISTANT APPLICATION
Als CUSTOMER JOURNEY EXPERT hou je je bezig met het bedenken van AI oplossingen, zoals bijvoorbeeld chatbots of virtuele assistenten. Ook ontwikkel je applicaties voor communicatie op de website.

FULL-STACK DEVELOPER
Als een FULL-STACK DEVELOPER ben je bezig met het ontwikkelen en updaten van Java gebaseerde webapplicaties. Ook kijk je samen met je team wat er nodig is wat voor specificaties er nodig zijn voor het product. Daarnaast hou je je ook bezig met de veiligheid van een applicatie.

Technieken binnen het bedrijf


Jenkins. (n.d.). [Logo]. https://www.jenkins.io/images/logo-title-opengraph.png

OUTSYTEMS
Outsystems is een low-code platform waarmee responsive applicaties mee gemaakt kan worden. Low-code wil zeggen dat je weinig programmeerkennis nodig hebt om zelf een applicatie te kunnen bouwen. Het voordeel van OutSystems is dat het een zeer korte ontwikkeltijd nodig heeft.

BLUE TEAM
Bij Blue Teaming staat de defensie centraal. Hierbij probeer je de security van een bedrijf te verbeteren. Een belangrijke taak is dat zij de aanval van een Red Team zo snel mogelijk ontdekken, zodat ze dan kunnen zien waar hun kwetsbaarheden zitten en dit direct oplossen.

JENKINS
Jenkins is een open source automatiseringsserver. Het helpt bij het automatiseren van de onderdelen van softwareontwikkeling die verband houden met het bouwen, testen en implementeren.

Bedrijfscultuur


ING Hoofdkantoor. (n.d.). [Photo]. https://www.banken.nl/illustrations/news/detail/2020-01-08-100808691-ING_Cedar_1.jpg

ING heeft een heel formele sfeer, de vacatures zijn allemaal in correct Engels en ook zie je alle medewerkers nette kleding dragen (meestal pak). Het bedrijf geeft aan dat ze heel divers zijn. Zo vinden ze het belangrijk dat ze een goede werkgever zijn. Ze hebben verschillende waarden en normen in het bedrijf. Dit leggen ze uit in de Orange Code.
