Capgemini

Capgemini

Capgemini is een bedrijf dat zijn werkgebied heeft in de ICT en consultancy. Het is een groot bedrijf met meer dan 270.000 medewerkers in meer dan 50 landen. De projecten die het bedrijf doet is heel divers, van ICT-diensten voor HEMA tot Eneco groep helpen met digitale transformatie en businessinnovatie.

It-rollen


Front-end developer. (n.d.). [Illustration]. https://miro.medium.com/max/17008/1*jBpjA63ARA3XBa-hTjyskg.jpeg

CYBER SECURITY CONSULTANT – SECURE SOFTWARE DEVELOPMENT
Als CYBER SECURITY CONSULTANT – SECURE SOFTWARE DEVELOPMENT ondersteun en adviseer je organisaties op zowel strategisch als organisatorisch niveau bij het beveiligen van hun applicaties.

TECHNICAL APPLICATION ENGINEER
Een TECHNICAL APPLICATION ENGINEER werkt aan applicaties met een groot maatschappelijk belang. Hij maakt deel uit van de business unit Application Managed Services (AMS). Bij storingen moet je goed en adequaat kunnen ingrijpen met het team.

SOFTWARE ARCHITECT (JAVA)
Als JAVA SOFTWARE ARCHITECT ben je bezig met het maken van geavanceerd maatwerkoplossingen in software voor klanten. Voornamelijk is hij bezig in Java.

FRONT-END DEVELOPER
Een FRONDEND DEVELOPER is voornamelijk bezig met het ontwikkelen en verbeteren van webapplicaties en mobile apps. Een FRONT-END DEVELOPER werkt samen met zijn team om tot een goed resultaat te komen.

DATA SCIENTIST
Als DATA SCIENTIST ben je bezig met het verzamelen en formuleren van data. Dit kan de data zijn van bijvoorbeeld sensoren maar ook van bijvoorbeeld trackingcookies. Je maakt de data voor de klant leesbaar. Je bent ook bezig met het gebruik van Machine learning met Python.

Technieken binnen het bedrijf


Front-end developer. (n.d.). [Illustration]. https://miro.medium.com/max/17008/1*jBpjA63ARA3XBa-hTjyskg.jpeg

AGILE
Agile is een manier van softwareontwikkeling. Er zijn verschillende ontwikkel methoden en frameworks, waaronder: Scurm, KanBan, etc. Door Agile te werken deel je zo je project op in kleine stukjes, en het is belangrijk om na elk stukje iets bruikbaars op te leveren.

Azure
Cloud computing-platform van Microsoft waarmee een aantal internetdiensten aangeboden kan worden via het internet of binnen de eigen omgeving van het bedrijf. De software die gebruikt wordt uit het Cloud systeem moet niet geïnstalleerd worden op de computer van de gebruiker.

JAVA

Java is een programmeertaal en computerplatform. Het is een platformonafhankelijke taal die qua syntax grotendeels gebaseerd is op C++. Veel applicaties en websites draaien op Java.

Bedrijfscultuur


Capgemini bedrijfspand. (n.d.). [Photograph]. https://daf9627eib4jq.cloudfront.net/app/uploads/2017/01/capgemini-1-1024x640.jpg

Bij Capgemini staat collegialiteit voorop. Ook is het multiculturele organisatie wat respect erg belangrijk vindt. In 2018 is Capgemini uitgeroepen tot World’s Most Ethical Companies, dit al voor het zesde jaar. Verder heeft het bedrijf door de jaren door veel awards en erkenningen gehad. De medewerkers geven aan dat het een open sfeer is waar iedereen elkaar wilt helpen. Ook krijg je veel vergoedingen. Capgemini heeft zeven gemeenschappelijke waarden: Eerlijkheid, Durf, Vertouwen, Vrijheid, Plezier, Bescheidenheid en Teamgeest.
