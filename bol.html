BOL.com

Bol.com

Bol.com heef zich in de laatste jaren ontwikkelt tot dé webshop van Nederland en België. In 2012 werd bol.com overgenomen door Ahold. In 2019 waren ca. 27.000 ondernemers aangesloten bij het bedrijf. Sinds 2011 kunnen ondernemers ook verkopen via bol.com doormiddel van het partnerprogramma. Er zijn meer dan 2.000 medewerkers in dienst bij het bedrijf.

It-rollen


Cloud Engineer. (n.d.). [Illustration]. https://searchcompany.nl/wp-content/uploads/2019/12/cloud_computing_engineer_header-1080x675.jpg

SECOPS ENGINEER
Als SECOPS ENIGNEER zorg je ervoor dat de data veilig is. Je houdt je ook bezig met het automatiseren van processen, werkt met het team aan beveiligingsoplossingen en houdt zich bezig met cloudbeveiliging.

SOFTWARE ENGINEER
Als SOFTWARE ENIGNEER hou je je bezig met het ontwikkelen van de software voor het bedrijf. Je werkt samen met je team om nieuwe features te ontwikkelen. Bij Bol.com heb je 5 verschillende afdelingen waar je in kan werken: Shopping Experience Space, Retail Platform Space, Logistics Space, Content & Marketing Space, Platform Space.

CLOUD ENGINEER
Als CLOUD ENGINEER ben je bezig met je team om de cloud en datacenter infrastructuur te runnen. Je ontwikkelt en update ook nieuwe features. Ook ben je verantwoordelijk voor de veiligheid van kernsystemen. Als CLOUD ENGINEER ben je ook veel bezig in teamverband.

API PLATFORM ENGINEER
Als API PLATFORM ENGINEER ben je de bouwer van APIs. Je houdt je bezig met het ontwikkelen, optimaliseren en integreren van API tools. Ook beoordeel en verbeter je API Guidelines en ondersteund teams bij API Ontwikkelprocessen.

SITE RELIABILITY ENGINEER
Als SITE RELIABILITY ENGINEER hou je je bezig met het logistieke systeem van het bedrijf. Je kijkt of je het systeem kan verbeteren. Maar een belangrijke taak is ook het signaleren van bedreigingen voor security.

Technieken binnen het bedrijf


Docker Architecture. (n.d.). [Illustration]. https://thingsolver.com/wp-content/uploads/pasted-image-0-768x452-1.png

DOCKER
Docker is een computerprogramma om het bestandssysteem van de computer te visualiseren. Docker wordt gebruikt om softwarepakketten uit te voeren (containers).

KOTLIN
Kotlin is een cross-platform programmeertaal, ontworpen om samen te werken met Java. Kotlin is vooral gericht op de Java Virtual Machine, maar kan ook complimeren naar JavaScript of machinecode.

NGINX

NGINX is een open source HTTP webserver, proxy mail server, and reverse proxy. NGINX staat bekend voor zijn hoge prestaties en laag geheugenverbruik.

Bedrijfscultuur


Bol.com Hoofdkantoor. (n.d.). [Photo]. https://banen.bol.com/content/uploads/2018/05/DJI_0004-1-1024x768.jpg

Bol.com is op zijn vacature website heel open over het bedrijf. De kantoren zijn licht en ruim. Het is een heel erg informele relaxte open sfeer. Ze willen een open en transparante werkomgeving waar iedereen elkaar kan aanspreken. Bol.com heeft verschillende waarden: Bevlogen, Samen, Vertrouwen, Oprecht en Nuchter.
